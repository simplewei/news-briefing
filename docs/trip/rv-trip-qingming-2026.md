# 🚐 杭州出发 · 清明节房车旅行计划
**日期**: 2026年4月4日 - 4月6日（3天2晚）
**成员**: 一家三口
**出发地**: 杭州

---

## 🗺️ 推荐路线：杭州 → 千岛湖 → 建德 → 新安江 → 杭州

> 环线设计，不走回头路，总里程约 **450km**，适合3天轻松游玩

### 为什么选这条线？

✅ 距离适中 — 单程2-3小时车程，不过度驾驶
✅ 房车营地成熟 — 千岛湖、建德都有房车停靠点
✅ 清明时节风景绝佳 — 油菜花、茶园、春江水暖
✅ 适合亲子 — 自然、人文、体验丰富
✅ 避开热门 — 比黄山、婺源人少很多

### 行程
| 时间 | 安排 | 备注 |
|------|------|------|
| 08:00 | 杭州出发 | 走杭千高速，约150km |
| 10:30 | 抵达千岛湖镇 | 下高速即到 |

## ⚠️ 注意事项

1. **清明天气** — 4月初浙西地区多雨，备好雨具
2. **新安江水温** — 常年14-17°C，漂流会冷，带换洗衣物
3. **营地预订** — 清明假期建议提前1周电话确认
4. **山路驾驶** — 环湖公路弯道多，注意车速
5. **充电规划** — 房车续航有限，规划好充电桩位置
