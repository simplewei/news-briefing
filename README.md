# 📰 NewsBrief — AI 新闻简报生成器

> 一句话生成专业新闻简报。选择领域、媒体源、推送时间 → AI 自动抓取、分析、生成。

## ✨ 核心能力

| 功能 | 说明 |
|------|------|
| 🎯 **多领域覆盖** | 地缘政治、科技、经济、能源、金融、健康、气候、国防 — 8大领域自由组合 |
| 🌐 **权威媒体源** | 新华社、Reuters、Bloomberg、BBC、FT 等 20+ 中英双语权威媒体 |
| ⏰ **定时推送** | 早间/午间/晚间/突发追踪/周报 — 自选节奏 |
| 🔍 **多源对比** | 同一事件，不同媒体如何报道？AI 横向对比 |
| 📊 **数据洞察** | 自动提取关键数据指标 |
| 🔮 **后续追踪** | 标注未来重要事件节点 |

## 🚀 快速开始

### 方式一：通过 OpenClaw 安装

```bash
clawhub install news-briefing
```

然后直接对 AI 说：
```
"帮我生成一份地缘政治新闻简报"
"每天早8点推送科技简报"
"经济+能源简报，中英双语，晚间推送"
```

### 方式二：手动使用

1. 克隆仓库
```bash
git clone https://github.com/simplewei/news-briefing.git
```

2. 配置 `config.json` 中的媒体源和领域

3. 对 AI 描述你的需求即可

## 📋 简报格式

每期简报固定包含 4 个板块：

```
┌─────────────────────────────────┐
│ 📋 今日要闻 (3-5条)             │
│ 来源 + 标题 + 一句话摘要         │
├─────────────────────────────────┤
│ 🔍 深度分析 (1-2个话题)         │
│ 多源对比，发现不同视角           │
├─────────────────────────────────┤
│ 📊 数据看点                     │
│ 关键指标表格，一目了然           │
├─────────────────────────────────┤
│ 📌 后续关注                     │
│ 未来 24-48h 值得追踪的事件      │
└─────────────────────────────────┘
```

## ⏰ 推送时间

| 类型 | 时间 | 场景 |
|------|------|------|
| 🌅 早间 | 08:00 | 开始一天前了解全局 |
| ☀️ 午间 | 12:00 | 午休快速浏览 |
| 🌆 晚间 | 18:00 | 收盘/下班后的每日总结 |
| 🚨 突发 | 每4h | 持续关注热点 |
| 📅 周报 | 周一 09:00 | 每周回顾与展望 |

## 🔧 自定义

编辑 `config.json` 添加自定义媒体源：

```json
{
  "mediaSources": {
    "zh": {
      "自定义分组": [
        {"name": "媒体名称", "searchHint": "site:example.com", "weight": 2}
      ]
    }
  }
}
```
